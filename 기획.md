# McJarSwap 기획
>Jar 파일 교체와 server.properties 수정, 서버 재시작 과정을 
터미널에서 반복하며 느낀 피로를 해결하고자, 직관적인 UI로 간편한
관리 도구를 개발하려는 프로젝트입니다.

### 기초 구상
